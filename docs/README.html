安装教程

  1. 下载好markdown系统， git clone git@gitee.com:xiaoxiyouran/markdown.git 然后 make 一下即可
  2. 生成的文档在 docs/top.html

使用说明

  1. 打开docs/top.html 即可。

  2. 如果编辑src文件夹下的markdown文件，如果有大型的代码或者软件，不想在生成docs 时拷贝时，请放在unmd文件夹下（若无，则手动创建这个文件夹）。
