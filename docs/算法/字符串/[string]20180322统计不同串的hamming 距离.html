题目如下，有一个长串S :aba, 则T 串 aa .串的长度范围是 1<= |T| <= |S| <= 10 ^5

aba                 aba
|                    |
aa(距离是1)           ab(距离是2)

在给一个更复杂一点的例子来说明我们的快捷方法：

abaaba          abaaba
|                |  
aa               aa .....

可以像上面这样暴力破解。但好像复杂度O(n) * O(m)

由于题目规定只有ab两种字符，可以编码为 0 和 1

010010      010010
|            |  
00           00  ......

最后简单的代码如下:

int main(){
	string s ;
	string t;
	cin >> s;
	cin >> t;

	int sum = 0;
	for( int i=0; i<s.size();i++ ){
		int head_s = i;
		int head_t = 0;
		for( int j = head_t; j < t.size();j++ ){
			if( s[head_s] != t[j] )
				sum++;
		}
	}

	cout << sum << endl;

	return 0;

}
